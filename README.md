#Dependencies
  - Jekyll 3.4.0 | `gem install jekyll`

# Local build
  - `git clone git@github.com:Kettul/kettul.github.io.git`
  - `cd kettul.github.io`
  - `bundle install`

  ##Run locally

  - To build site files - `bundle exec jekyll build`
  - To build and run on localhost:4000 - `bundle exec jekyll serve`
